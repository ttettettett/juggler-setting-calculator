# juggler-setting-calculator

このアプリは、ジャグラー設定判別ツールです。ユーザーは、特定のジャグラー機種でのゲーム数、ビッグボーナス（BB）回数、レギュラーボーナス（RB）回数を入力することで、設定の判別ができるようになっています。アプリは、簡単な操作でBBやRBをカウントアップするボタンも提供しています。

以下の機能が含まれています:

ジャグラー機種の選択: ユーザーは、プルダウンメニューから機種を選択できます。選択された機種に応じて、設定判別の基準が変わります。
ゲーム数、BB回数、RB回数の入力: ユーザーは、これらの値を入力欄に手動で入力できます。
カウントアップボタン: ゲーム数、BB回数、RB回数を簡単にカウントアップできるボタンが用意されています。
設定判別: 入力された値に基づいて、各確率と設定を計算し、表示します。設定は1から6までの範囲で判定されます。
このツールは、ジャグラーの設定を推測し、遊戯者が現在の状況を把握するのに役立ちます。また、アプリはレスポンシブデザインで作成されており、デバイスの画面サイズに応じてレイアウトが調整されます。
